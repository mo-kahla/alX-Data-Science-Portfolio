# alX-Data-Science-Portfolio


Welcome to my data science portfolio! This repository showcases my projects completed during the 13-month Data Science program by ALX Africa.

## About the Program

The ALX Data Science programme, developed in partnership with ExploreAI includes:

- **Data Science Specialisation**: Covers spreadsheets, statistics, data modelling, SQL, data visualisation with Power BI, and programming with Python.
- **Cloud Computing**: The final module focuses on applying data science skills in cloud computing.

[Link to the alX Data Science programme](https://www.alxafrica.com/programme/data-science/)

[Link to the alX Data Science Curriculum](https://www.alxafrica.com/wp-content/uploads/2023/02/ALX-Data-Science-Curriculum.pdf)
